ABOUT

CSE NotesHUB is a online notes portal. This is a Mini-Project for BE- Computer Science 5th semester in the year of 2021-2022.
This web application is useful for university students and professors of CSE department, for storing all subject module notes, question paper, syllabus copy and many other related things in a well-organised way. This application also allows the admin to modify, upload and delete the data from the database.

This web application was coded using
Frontend: HTML5, CSS3, JavaScript
Framework: Bootstrap 5
Database: MySQL
Backend: Python, Flask
